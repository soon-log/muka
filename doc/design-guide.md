# MUKA Design Guide v1.0

## 브랜드
- 무드: 따뜻하고 감성적인
- 키워드: 따뜻함, 친근함, 감성적, 플레이풀, 귀여움

## 로고
- 형태: "muka" 영문 텍스트
- 스타일: 커스텀 레터링, 플레이풀/귀여운, 통통한 글자
- 컬러: 코랄 (단일 고정)
- 위치: 카드 하단 우측, 작게

### 카드 내 텍스트
- 질문: 18-20px, Regular
- 곡명: 16-18px, Bold
- 아티스트: 14px, Regular, 투명도 70%

## 카드 디자인

### 규격
- 비율: 9:16
- 크기: 1080x1920px

### 레이아웃
```
[질문]
[앨범커버 50-60%]
[곡명]
[아티스트]
[로고 - 우하단]
```

### 앨범커버
- 크기: 카드 너비 50-60%
- 스타일: 원본 그대로 (효과 없음)

## 카드 7종 무드

| # | 무드 | 컬러 |
|---|------|------|
| 1 | 밝음/경쾌 | 노란색, 오렌지 |

| 3 | 몽환적 | 라벤더, 연보라 |
| 4 | 따뜻함 | 피치, 코랄 |
| 5 | 레트로 | 머스타드, 브라운 |
| 6 | 감성적/쓸쓸 | 그레이, 블루그레이 |
| 7 | 로맨틱 | 핑크, 로즈 |

## UI 컴포넌트

### 버튼
- Primary: 코랄 배경, 흰색 텍스트, radius 12px
- Secondary: 연한그레이 배경, 다크그레이 텍스트, 1px 테두리

### 검색창
- 흰색 배경, 1px 테두리, radius 12px
- 포커스: 코랄 테두리
- 좌측 돋보기 아이콘

### 카드 선택기
- 스와이프 또는 좌우 화살표
- 인디케이터: 점 7개 (활성=코랄)

### 바텀시트
- 흰색 배경, 상단만 radius 20px
- 상단 그레이 핸들바

## 화면 구성

### 질문 선택 (A)
- 헤더: 로고
- 안내 텍스트
- 질문 카드 리스트 (7개)

### 음악 검색 (B)
- 헤더: 뒤로가기 + 질문
- 검색창 (상단 고정)
- 검색 결과 리스트 (앨범커버+곡명+아티스트)

### 카드 미리보기 (B)
- 헤더: 뒤로가기 + "카드 선택"
- 카드 (스와이프)
- 인디케이터
- 공유하기 버튼

### 카드 확인 (A)
- 헤더: 로고
- 카드
- 이미지 저장 버튼
- 음악 듣기 버튼 → 바텀시트

## 반응형
- 모바일 기준: 375px
- 데스크탑: 모바일뷰 유지, 중앙정렬 (max-width 430px)

## OG 이미지
- 크기: 1200x630px
- 구성: 로고 + 질문 + 카드미리보기 + 안내문구
- 질문별 동적 생성
