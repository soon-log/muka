# MUKA PRD v1.0

## 개요
- 서비스명: 뮤카(Muka)
- 컨셉: A가 친구 B에게 음악 추천 요청 → B가 음악 선택 후 카드로 응답
- 핵심가치: 알고리즘 아닌 친구가 고른 음악, 타인의 시선으로 음악 발견
- 사용동기: 호기심/재미, 추천 요청

## 사용자 플로우

### A(발신자)
1. 앱 접속 (로그인 불필요)
2. 질문 템플릿 7개 중 선택
3. 해시 링크 생성
4. 카카오톡으로 B에게 전송
5. B로부터 카드 링크 수신
6. 카드 확인 → 스트리밍 앱 이동 or 이미지 저장

### B(수신자)
1. 카카오톡으로 링크 수신
2. 링크 클릭 → 웹페이지
3. 음악 검색 (자동완성)
4. 곡 1개 선택
5. 카드 디자인 확인/변경 (스와이프)
6. 공유 버튼 → 카카오톡으로 A에게 전송

## 질문 템플릿 (7개)
1. 나한테 어울리는 음악은? (관계/A중심)
2. 내가 힘들 때 들으면 좋을 음악은? (관계/A중심)
3. 나를 생각하면 떠오르는 음악은? (관계/B중심)
4. 요즘 네가 꽂힌 음악은? (상황/가벼움)
5. 아무 생각 없이 들을 수 있는 음악 추천해줘 (상황/가벼움)
6. 인생 노래 하나만 추천해줘 (상황/진지함)
7. 비 오는 날 듣기 좋은 음악은? (상황/진지함)

## 핵심 기능
- 일회성 링크: 해시 기반, 1회 응답 후 폐기, 7일 만료
- 음악 검색: Spotify API, 로그인 불필요, 1곡만 선택
- 카드: 7종 랜덤 배정, 스와이프로 변경 가능, 9:16 비율
- 공유: 카카오톡 공유 API + 링크 복사
- 저장: 카드 이미지 저장 (질문+카드+로고 포함)
- 스트리밍: 바텀시트에서 플랫폼 선택 (Spotify, YouTube Music, 멜론)

## MVP 범위
